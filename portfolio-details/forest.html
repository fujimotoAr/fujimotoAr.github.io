U-Net and Fully Convolutional Network for Forest Segmentation: A Comparative Study

Background


The increasing occurrence of deforestation activities negatively affects the environment in Indonesia. Among these impacts are global warming and the destruction of wildlife habitat. With the advances in technology, particularly in aerial imaging with drones and satellites, we can analyse the aerial images to map forest and non-forest segments given an image to track the deforestation rate. In this project we compared U-Net and Fully Convolutional Network (FCN) architectures for forest segmentation. We use two different datasets for this project, which is "Amazon" and "Forest Aerial Images for Segmentation" datasets.

Objective


This project was aimed to:

  1. To perform forest image segmentation.
  2. To find the best method to segment forest images.

Dataset


We use two datasets in this project: which is the Amazon dataset [1] containing 45 images with each image sized 512*512 pixels with three colour channels, and the Forest Aerial Images for Segmentation dataset [2] containing 5108 images sized 256*256 pixels with three colour channels. Each image from both datasets comes in pair with its corresponding masks which functions as its label. Model Architecture


The U-Net architecture consists of contracting and expansive paths, as can be seen in the picture below. The contracting path consists of convulutional and pooling layers like those in Convolutional Neural Networks (CNN). On the other hand, an upsampling process is done in the expansive path, followed by up-convolutions, merging of feature maps from the contracting path, and also repeated usage of convolutional layers [3].


U-Net architecture [3]


For the Fully Convolutional Network (FCN) architecture, there are several variants of FCN, among those are the FCN-32, FCN-16, and FCN-8. The main difference between these variants is the number of upsampling processes, which is one for FCN-32, two for FCN-16, and three for FCN-8. In this project, we used only FCN-16 and FCN-8 to perform forest image segmentation. An illustration for FCN and its variants is shown in a picture below.


Fully Convolutional Networks (FCN) and its variants [4]


Evaluation


In this project, we use three metrics to evaluate segmentation performance for both U-Net, FCN-16 and FCN-8 architectures. We use Intersection over Union, Dice coefficient, and Accuracy metrics.

Results


The segmentation performance based on Intersection over Union, Dice, and Accuracy for the three architectures can be seen in the table below.


Evaluation results for all architectures with both datasets


Based on these results, we can conclude that the usage of FCN-8 architecture results in better segmentation performance for all three metrics in both datasets. On the Amazon dataset, FCN-8 came out on top for all metrics, followed by U-Net and FCN-16. On the other hand, for Forest Aerial Images for Segmentation dataset, FCN-16 produced better performance than U-Net.
