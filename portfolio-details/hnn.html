Hierarchical Image Classification

Hierarchical Image Classification with Hierarchical Neural Network with CIFAR-100 dataset

Background


This project is based on my undergraduate thesis titled "Comparative Study between Hierarchical Neural Network and Single Convolutional Neural Network for Image Classification with Hierarchical Data". This project compares image classification performance for both hierarchical (HNN) and single CNN models. Both models are evaluated based on Top-1 Accuracy and F1-Score.

Objective


This project is aimed to:

  1. To compare the classification performence between HNN and single CNN models on CIFAR-100 dataset subset.

Dataset


Both CNN and HNN models are trained on a subset of CIFAR-100 dataset, which is a dataset with hierarchical class labels. 10 out of 20 coarse classes in the base dataset are used in this project, with each coarse class containing 5 fine classes. There are 30000 images in total which are used for both training and validation phase for both models. Each image on the dataset is a 32*32 resolution image with three color channels. Model Architecture


VGG-16 pre-trained model with modified multi layer perceptron and output layer is used for both architecture, which is 1 model for CNN and 11 (10 models for fine-level classification and 1 model for coarse-level classification). The main differences between CNN and HNN architectures are the number of steps to produce a final classification label given an image, and the fact that a CNN model only produces fine-level classification model, rather than hierarchical (both coarse and fine levels) classification as performed by HNN. Evaluation


In this project, we use two different metrics to evaluate image classification performance for both CNN and HNN models. We use Top-1 Accuracy metric as the main evaluation metric, and also F1-Score metric as a secondary metric.

Based on these results, we can conclude that the differences of classification performance between single CNN and HNN models are not very significant, with the results slightly favor that of single CNN model. For both models, the Top-1 Accuracy score tops out at around 52%. We suspect that the relatively small size of images and also that the amount of images for each classes is relatively few might contribute to the low classification performance. For further developments, it is suggested to increase the number of images by combining this dataset with other relevant datasets.
