Indonesian Nickel Export Data

Database and visualization of nickel commodities export

Background


For this project, we implemented a python program to retrieve monthly data related to nickel exports from Indonesia to other countries from the UN Comtrade website using the provided free API in our python implementation. I will give a summary for data visualization using Tableau with a subset data used in this project, which is nickel exports from 2015 to 2019.

Objective


For this project, we aimed to:

  1. To retrieve monthly nickel exports data from UN Comtrade website
  2. To store the fetched data to a MySQL database

Data Visualization


We use Tableau to create visualisasion from the data stored in a MySQL database. For our bar chart, we opted to use a logarithmic scale as the export quantity of each country varies greatly, in which case a logaritmic chart scale makes the chart easier to read.
